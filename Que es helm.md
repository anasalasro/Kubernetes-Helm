# ¿Qué es helm? ¿Qué es un chart?

``` ruby
Helm (del término marítimo de timón) es una herramienta para gestionar aplicaciones de Kubernetes.  
Helm te ayuda a timonear Kubernetes usando cartas de navegación, en inglés se conocen como Helm Charts. 

Helm usa un formato de empaquetado llamado charts.  
Un chart es una colección de archivos que describen un conjunto relacionado de recursos de Kubernetes.  
Se puede usar un solo chart para implementar algo simple, como un pod de Memcached, o algo complejo, 
como una pila de aplicaciones web completa con servidores HTTP, bases de datos, cachés, etc.

```
